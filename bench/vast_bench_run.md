# vast_bench_run.sh — Vast.ai Benchmark Automation

One-shot script: rent GPU → deploy → benchmark Mixtral-8x7B on ShareGPT → retrieve results → destroy.

## Prerequisites

- **vastai CLI** — `pip install vastai`
- **jq** — `apt install jq` or `brew install jq`
- `ssh` / `scp` — standard openssh-client

## How to Run

```bash
# Basic run (default: Mixtral-8x7B-Instruct-v0.1, TP=2)
bash bench/vast_bench_run.sh

# Custom model / tensor parallelism
bash bench/vast_bench_run.sh --model mistralai/Mixtral-8x7B-v0.1 --tp 2

# Preview all steps without executing
bash bench/vast_bench_run.sh --dry-run
```

The script will prompt twice before spending money: once to destroy any existing
instances, once to confirm the selected offer.

## Cost Estimate

| Config | Duration | Approx cost |
|--------|----------|-------------|
| 2× A100 40 GB | ~2 hr | $6–10 |
| 2× A100 80 GB | ~2 hr | $9–14 |
| 2× RTX 3090 | ~3 hr | $4–7 |

Instances are destroyed immediately after results are retrieved.
The EXIT trap also destroys the instance if the script fails mid-run.

## Output

Results land in `bench/results/vast-YYYYMMDD-HHMMSS/`:

```
summary.json          # top-level metrics (throughput, dispatch rate, …)
phase1_*.json         # per-concurrency latency/throughput sweep
charts/               # PNG plots generated by plot_results.py
```

## Re-running Charts on Existing Results

```bash
python bench/charts/plot_results.py \
  --results-dir bench/results/vast-20260520-153000 \
  --output-dir  bench/results/vast-20260520-153000/charts
```
